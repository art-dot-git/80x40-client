80x40 is an 80 character wide by 40 character tall block of text stored in a git repo. A script watches the repo and merges in any pull requests that change the contents of this block. It is effectively a collaborative ASCII art canvas that anyone with a Github account can draw on.

The block above shows the current version of the block. To make changes, simply fork the git repo, edit the text block locally, and submit a pull request.
